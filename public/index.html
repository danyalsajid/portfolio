Restricted Area Maps

Previous Next

Demo | GitHub

An app from which a user can perform his analysis for which places are restricted to open new stores and how much restricted radius they have along with an interactive menu to further enhance your findings.
Implementation details:
I wrote a web scraper program to read the location data of stores from specific websites. Using this data and google maps geocoding API I extracted the coordinates of these locations. This data was stored on a server from where the client will make the request to fetch that data. Now in the client side, using google maps reverse geocoding API and places API I extracted the location and placed the marker on the map. This marker has a green circular radius around it with a variable distance to mark the vicinity of that area. The red circles are the public places whose radius is also variable. Selecting certain options from the menu on the right will give you the required results to perform your analysis.
Tech Stack Used:
  • Client Side: Angular, Bootstrap, Google Maps API
  • Server Side: Node, Cheerio, Firebase, Google Maps API
